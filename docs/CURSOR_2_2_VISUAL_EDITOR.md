# Cursor 2.2 視覺化編輯器更新 / Cursor 2.2 Visual Editor Update

## ✨ 快速摘要

- Cursor Browser 內嵌**視覺化編輯器**，同一視窗同時檢視頁面渲染、程式碼庫與控制面板。
- 支援直接在頁面上**拖拉調整 DOM 階層與排序**，比對不同格線或區塊排列後再套用到程式碼。
- 透過側邊欄集中**元件屬性/狀態與樣式控制**（色彩 Token、字體、間距滑桿、即時取色器）。
- **Point & Prompt**：點選元素並輸入自然語言指令，代理並行處理多個修改並寫回對應檔案。

## 🧩 核心體驗

| 能力 | 說明 | 帶來的好處 |
| ---- | ---- | ---------- |
| 視覺 + 代碼同窗 | 頁面渲染、程式碼庫、視覺化工具並列 | 減少上下文切換、即時確認變更 |
| 拖拉 DOM 重排 | 直接拖放按鈕、區塊或格線調整順序與階層 | 快速測試版面、驗證互動優先級 |
| React 變體切換 | 側邊欄整理元件屬性/狀態，快速切換尺寸、配色、互動模式 | 加速比較多種元件變體 |
| 樣式微調 | 色彩 Token、調色盤、滑桿、即時取色器 | 與設計系統對齊、可視化調整 |
| Point & Prompt | 點選元素 + 文本描述形狀/顏色/排序等需求 | 以自然語言驅動修改，減少手動編輯 |

## 🔄 代理寫回流程

1. 在視覺化編輯器拖拉或切換屬性，確認畫面符合預期。
2. 以提示詞要求代理套用變更；代理會定位對應元件與相關檔案。
3. 代理將調整寫回程式碼層，可並行處理多個修改需求。

## 🆕 其他 2.2 版本更新

- **除錯模式**：利用執行期紀錄重現問題並追蹤原因。
- **計畫模式升級**：支援嵌入 Mermaid 圖表，並可將選定待辦分派給新代理。
- **多代理評判**：並行代理完成後自動比較結果，給出推薦解並附上選擇理由。
