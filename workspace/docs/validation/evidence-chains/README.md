# 證據鏈存儲目錄

此目錄用於存儲驗證系統生成的不可變證據鏈。

## 文件類型

- `sha256_evidence.db` - SHA-256 哈希證據數據庫
- `sha512_evidence.db` - SHA-512 哈希證據數據庫
- `quantum_signatures.db` - 量子簽名數據庫

## 保留策略

- 證據文件保留 90 天
- 自動歸檔至長期存儲

## 安全性

- 所有證據文件均為只讀
- 使用 BLAKE3 進行完整性驗證
- 支持後量子密碼簽名

## 注意事項

- 此目錄中的文件由系統自動生成
- 請勿手動修改或刪除文件
- 如需清理，請使用官方維護腳本
